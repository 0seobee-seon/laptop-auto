야근수당내역 파일을 자동 생성하는 명령어입니다.

## 사용법
```
/야근수당 [1번파일경로] [부서명] [연도] [월]
```
예시:
- `/야근수당 "C:\업무\안전진단_5월.xlsx" 안전진단팀 2026 05`
- `/야근수당` (인수 없이 실행하면 대화형으로 진행)

---

## 실행 절차

### 1단계 — 인수 파싱
`$ARGUMENTS`를 공백 기준으로 파싱합니다 (경로에 공백이 있으면 따옴표로 감쌈).
- 인수 1: 1번 야근현황 파일 경로 (xlsx/xls)
- 인수 2: 부서명
- 인수 3: 연도 (4자리)
- 인수 4: 월 (2자리, 예: 05)

인수가 부족하면 한국어로 질문해서 받아오세요.

### 2단계 — 핵심 스크립트 위치 확인
아래 Python 코드를 실행해 `야근수당_자동입력.py` 파일을 찾으세요.

```python
import os, sys

# 탐색 우선순위: 1번파일과 같은 폴더 → 현재 작업 디렉토리
src_path = "<1번파일경로>"
candidates = [
    os.path.dirname(os.path.abspath(src_path)),
    os.getcwd(),
]
core_script = None
for d in candidates:
    p = os.path.join(d, "야근수당_자동입력.py")
    if os.path.exists(p):
        core_script = p
        break

if core_script is None:
    print("ERROR: 야근수당_자동입력.py 파일을 찾을 수 없습니다.")
    print("1번 파일과 같은 폴더에 야근수당_자동입력.py 가 있어야 합니다.")
else:
    print(f"스크립트 위치: {core_script}")
```

파일을 찾지 못하면 사용자에게 `야근수당_자동입력.py`의 위치를 물어보고 경로를 받아 진행하세요.

### 3단계 — 야근수당내역 생성
스크립트를 찾았으면 아래 코드를 실행하세요.

```python
import importlib.util, pandas as pd, os

# core 로드
spec = importlib.util.spec_from_file_location("core", core_script)
core = importlib.util.module_from_spec(spec)
core.__name__ = "core"
spec.loader.exec_module(core)

src        = "<1번파일경로>"
dept       = "<부서명>"
year       = "<연도>"
month      = "<월>"
sheet_name = f"{year}.{month}월"
out_dir    = os.path.dirname(os.path.abspath(src))

xf = pd.ExcelFile(src)
data_sheets = [s for s in xf.sheet_names if s not in core.SKIP_SHEETS]
core.DEPT_MAP     = {s: dept for s in data_sheets}
core.SPECIAL_DEPT = {}

emp = core.extract_overtime(src)
print(f"추출된 직원: {len(emp)}명")

output = os.path.join(out_dir, f"{dept}_야근수당내역_{year}_{month}월.xlsx")
core.create_new(emp, sheet_name, output)
print(f"저장 완료: {output}")
```
